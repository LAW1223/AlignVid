# AlignVid — Project Page

This branch (`gh-pages`) hosts the project page for **AlignVid (ICML 2026)**:
*Taming Visual Dominance via Training-Free Attention Modulation in Text-guided
Image-to-Video Generation.*

🌐 **Live site:** <https://law1223.github.io/AlignVid/>

Useful links:
- 📄 Paper: <https://arxiv.org/abs/2512.01334>
- 💻 Code: <https://github.com/LAW1223/AlignVid>
- 🤗 Benchmark: <https://huggingface.co/datasets/AIPeanutman/OmitI2V>

Page is a static site (Bulma + a small JS for the video-comparison grid).
To preview locally, just open `index.html` in a browser.
